## 更新规则（强制）

- RFC 状态任意变更时（Draft / Review / Approved / Implementing / Done / Blocked），必须在同一提交更新本表。
- 任何涉及迁移、重构、删改功能的合并到 `main`，必须在同一提交更新本表。
- 处于 Implementing 状态的 RFC，按 Asia/Shanghai 自然日每天至少更新一次。

## 状态说明

- Draft：起草中
- Review：评审中
- Approved：已通过，可开始编码
- Implementing：编码中
- Done：完成并合并
- Blocked：阻塞中
