# 迁移进度（progress）

> 记录粒度：以 RFC 为单位。每个 RFC 通过评审后才能进入编码状态。

## 状态说明

- Draft：起草中
- Review：评审中
- Approved：已通过，可开始编码
- Implementing：编码中（必须已 Approved）
- Done：完成并合并
- Blocked：阻塞（写清原因）
