## 四、横向发现

1. **卖出/回避信号是这 18 个模型的主要价值**。负向组合 19/19 在 HOLDOUT 保持同方向且多数放大（2024-2026 区间放大了下跌信号的边际）。A 股缺乏低成本做空手段，这些信号的实用落点是持仓退出、TPSL 与买入否决（与平台"同日负向优先退出并否决买入"的规则一致），而非做空获利。
2. **买入信号整体薄弱**。8 个正向组合中仅 1 个（S0000+S0004）事件级与组合级同时在样本外为正；最显著的 S0005 系 ENGULFING（q=0.019）事件边际保持但组合未兑现（入场拥挤度/持仓上限约束下 alpha 不足以覆盖调仓损耗）。
3. **统计强度普遍有限**。27 个组合中 VALIDATION q<0.05 的只有 3 个（rank 8/18/22，其中两个方向还不稳定）；其余多在 0.05~0.2 区间（门槛即 0.2）。单看任何一个模型都不足以支撑独立实盘策略。
4. **触发类型下钻有信息量**：MA5_TURN 在 S0000/S0005/S0009 上普遍优于全触发基线；PRICE_VOLUME_CONFIRMATION 在所有出现处（S0008、S0016）都表现出跨段变号，应视为不可靠触发。
5. **6 个模型（S0001/S0007/S0011/S0015 及派生 S0013/S0014）在本口径下无证据**，评价为"未证明"而非"无效"——它们未通过 TRAIN 门槛，可能在其他 horizon/参数下需另行研究。

## 五、限制与披露

- HOLDOUT 仅一段（2024-01-02～2026-07-21）单次揭示，负向边际的放大可能吃了该区间弱市的 beta；不构成对未来的保证。
- RAW 撮合不含分红送转现金流（CORPORATE_ACTION_NOT_FULLY_LEDGERED 为研究近似）；组合最差年份多为 2026（区间截断）。
- 排行的 `year_positive_ratio/worst_year` 按原始收益计，负向组合读数需按方向翻转解读。
- 本报告为揭示后的只读派生分析，未改变任何 artifact、ledger 或冻结状态；数字可由 run manifest 与 API 逐条复核。
