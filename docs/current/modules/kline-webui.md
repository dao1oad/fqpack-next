# Kline Web UI

## 职责

负责 KlineSlim 与相关图表页面展示。

## 入口

- Vue Kline 页面
- 后端 `/api/stock_data` 及相关结构接口

## 依赖

- 历史/实时行情
- 缠论结构接口

## 数据流

API -> Kline 页面 -> 多周期图层与结构面板

## 存储

依赖行情与结构计算结果，不持有独立事实源。

## 配置

关注 symbol 默认值、多周期显示、缩放/平移语义。

## 部署/运行

改动后通常影响 Web UI 与 stock_data 路由。

## 排障点

- 页面空白
- 图层残影
- 多周期结构错位
