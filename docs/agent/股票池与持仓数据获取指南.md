# 股票池与持仓数据获取指南

## 概述

FreshQuant 维护三类股票池数据和一个持仓数据集合，用于选股、信号生成和实盘交易。

## 数据类型与集合

| 类型 | MongoDB 集合 | 用途 |
|------|-------------|------|
| 候选股票池 | `stock_pre_pools` | 信号筛选后的初步股票池，含止损价和过期时间 |
| 股票池 | `stock_pools` | 正式股票池（自选股），未过期记录 |
| 必选股票池 | `must_pool` | 必须监控的股票/ETF |
| 持仓数据 | `stock_fills` | 交易成交记录，用于计算持仓 |

## 1. 候选股票池 (stock_pre_pools)

### 数据结构

```javascript
{
  _id: ObjectId,
  code: "000001",              // 股票代码（6位）
  name: "平安银行",            // 股票名称
  category: "CLX10001",        // 分类/信号类型
  datetime: ISODate,           // 入池时间
  stop_loss_price: 10.50,      // 止损价格
  expire_at: ISODate,          // 过期时间（默认89天）
  extra: {                     // 额外字段（扩展用）
    amount: 123456789,         // 成交额等
    ...
  }
}
```

### 保存候选股票

**函数**：`save_a_stock_pre_pools`

**来源**：`freshquant.signal.a_stock_common`

```python
from freshquant.signal.a_stock_common import save_a_stock_pre_pools
from datetime import datetime

# 基本用法
save_a_stock_pre_pools(
    code="000001",
    category="CLX10001",           # 信号类型/分类
    dt=datetime.now(),             # 入池时间
    stop_loss_price=10.50,         # 止损价格
    expire_at=datetime.now() + timedelta(days=89)  # 过期时间
)

# 带额外字段
save_a_stock_pre_pools(
    code="000001",
    category="CLX10001",
    stop_loss_price=10.50,
    amount=123456789,              # 成交额等额外字段
    custom_field="custom_value"    # 自定义字段
)
```

### 查询候选股票池

**函数**：`query_a_stock_pre_pool`

**来源**：`freshquant.data.astock.pre_pool`

```python
from freshquant.data.astock.pre_pool import query_a_stock_pre_pool

# 查询指定分类
records = query_a_stock_pre_pool(categories=["CLX10001", "CLX10002"])

# 查询所有
all_records = query_a_stock_pre_pool()

# 按代码查询
records = query_a_stock_pre_pool(codes=["000001", "000002"])
```

### 批量导入

```python
from freshquant.data.astock.pre_pool import save_a_stock_pre_pool

# 覆盖模式（清空后再添加）
save_a_stock_pre_pool(
    codes=["000001", "000002"],
    category="CLX10001",
    stop_loss_price=10.50,
    append_mode=False  # False=覆盖，True=追加
)
```

### 删除候选股票

```python
from freshquant.data.astock.pre_pool import remove

# 删除指定分类
remove(category="CLX10001")

# 删除指定股票
remove(codes=["000001", "000002"])

# 删除分类中的指定股票
remove(category="CLX10001", codes=["000001"])

# 按 ID 删除
remove(id="507f1f77bcf86cd799439011")
```

## 2. 股票池 (stock_pools)

### 数据结构

```javascript
{
  _id: ObjectId,
  code: "000001",              // 股票代码
  name: "平安银行",            // 股票名称
  category: "自选股",          // 分类名称
  datetime: ISODate,           // 入池时间
  stop_loss_price: 10.50,      // 止损价格（仅当为空时更新）
  expire_at: ISODate,          // 过期时间（默认10天）
  extra: {}                    // 额外字段
}
```

### 保存股票池

**函数**：`save_a_stock_pools`

**来源**：`freshquant.signal.a_stock_common`

```python
from freshquant.signal.a_stock_common import save_a_stock_pools

save_a_stock_pools(
    code="000001",
    category="自选股",
    dt=datetime.now(),
    stop_loss_price=10.50,
    expire_at=datetime.now() + timedelta(days=10)
)
```

**注意**：`stop_loss_price` 仅在现有记录为空时更新，不会覆盖已有值。

### 查询股票池

**函数**：`get_a_stock_pools`

**来源**：`freshquant.signal.a_stock_common`

```python
from freshquant.signal.a_stock_common import get_a_stock_pools

# 获取所有未过期的股票
pools = get_a_stock_pools()

# 返回列表，每个元素为字典
for pool in pools:
    print(pool['code'], pool['name'], pool['category'])
```

### 获取股票池代码

**函数**：`get_stock_pool_codes`

**来源**：`freshquant.data.astock.pool`

```python
from freshquant.data.astock.pool import get_stock_pool_codes

# 返回去重的代码列表
codes = get_stock_pool_codes()  # ['000001', '000002', ...]
```

### 删除股票池

```python
from freshquant.data.astock.pool import remove

# 删除指定分类
remove(category="自选股")

# 删除指定股票
remove(codes=["000001"])
```

## 4. 持仓数据 (stock_fills)

### 数据结构

```javascript
{
  _id: ObjectId,
  op: "买入" | "卖出",          // 操作
  symbol: "sz000001",          // 股票代码（带市场）
  stock_code: "000001",        // 纯代码
  name: "平安银行",            // 名称
  date: 20240101,              // 日期（YYYYMMDD）
  time: "09:30:00",            // 时间
  price: 10.50,                // 成交价
  quantity: 100,               // 数量
  amount: 1050.00,             // 金额
  amount_adjust: 1.0,          // 调整系数
  source: "trade"              // 来源
}
```

### 获取持仓列表

**函数**：`get_stock_positions`

**来源**：`freshquant.data.astock.holding`

```python
from freshquant.data.astock.holding import get_stock_positions

# 获取所有持仓（持仓数量>0 或 持仓金额<0）
positions = get_stock_positions()

for pos in positions:
    print(pos['symbol'], pos['quantity'], pos['amount'])
```

**持仓筛选条件**：
- `quantity > 0` （持有数量为正）
- `amount_adjusted < 0` （投入金额为负，表示成本）

### 获取持仓代码

**函数**：`get_stock_holding_codes`

**来源**：`freshquant.data.astock.holding`

```python
from freshquant.data.astock.holding import get_stock_holding_codes

# 返回 6 位代码列表（带 Redis 缓存）
codes = get_stock_holding_codes()  # ['000001', '000002', ...]
```

### 获取单个股票持仓

**函数**：`get_stock_hold_position`

**来源**：`freshquant.data.astock.holding`

```python
from freshquant.data.astock.holding import get_stock_hold_position

position = get_stock_hold_position("sz000001")
if position:
    print(f"持仓数量: {position['quantity']}")
    print(f"持仓金额: {position['amount']}")
```

## 5. 监控代码集合

### 获取监控代码

**函数**：`get_stock_monitor_codes`

**来源**：`freshquant.data.astock.pool`

```python
from freshquant.data.astock.pool import get_stock_monitor_codes

# 获取所有需要监控的代码（持仓 + 必选池 + 股票池）
codes = get_stock_monitor_codes()

# 仅获取持仓代码
codes = get_stock_monitor_codes(holding_only=True)
```

**监控代码组成**：
```python
holding_codes + must_pool_codes + stock_pool_codes (去重)
```

## 完整示例

### 选股后保存到候选池

```python
# freshquant/screening/clxs.py
from freshquant.signal.a_stock_common import save_a_stock_pre_pools
from datetime import datetime

# 选股结果
for index, row in result_df.iterrows():
    code = row['code']
    dt = row['date']  # date 类型
    stop_loss_price = row['stop_loss_price']

    # 保存到候选池
    save_a_stock_pre_pools(
        code=code,
        category="CLX10001",
        dt=datetime.combine(dt, datetime.min.time()),
        stop_loss_price=stop_loss_price,
        amount=row['amount']
    )
```

### 查询所有监控代码

```python
from freshquant.data.astock.pool import get_stock_monitor_codes

# 获取需要监控的所有代码
codes = get_stock_monitor_codes()

# 用于后续行情订阅或信号计算
for code in codes:
    # 订阅行情或计算信号
    pass
```

### 持仓分析

```python
from freshquant.data.astock.holding import get_stock_positions, get_stock_holding_codes

# 获取持仓代码
holding_codes = get_stock_holding_codes()

# 获取持仓详情
positions = get_stock_positions()
for pos in positions:
    print(f"股票: {pos['name']} ({pos['symbol']})")
    print(f"数量: {pos['quantity']}")
    print(f"成本: {pos['amount']}")
```

## 注意事项

1. **过期时间**：候选池默认 89 天，股票池默认 10 天
2. **止损价格**：股票池的止损价只在为空时更新，不会覆盖
3. **代码格式**：持仓使用带市场代码（sz000001），候选池/股票池使用纯代码（000001）
4. **Redis 缓存**：部分查询函数有缓存，注意更新后可能需要等待缓存过期
5. **extra 字段**：候选池和股票池都支持扩展字段，用于存储自定义数据
