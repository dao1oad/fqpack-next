# 持仓标的买卖点扫描器

## 功能

扫描持仓（A股 + ETF）在日线和30分钟线上的买卖点信号。

- **自动识别标的类型**（A股或ETF），使用相应的数据源获取行情数据
- **支持指定日期检测**，可回测历史某一天的信号情况
- 默认检测最后一个交易日的信号

## 使用方法

```bash
# 扫描最后一个交易日的信号（默认）
python -m script.holding_signals_scanner.scanner

# 扫描指定日期的信号
python -m script.holding_signals_scanner.scanner --date 2024-01-15

# 扫描昨天的信号
python -m script.holding_signals_scanner.scanner --date yesterday
```

## 命令行参数

| 参数 | 短参数 | 说明 | 格式 |
|------|--------|------|------|
| `--date` | `-d` | 目标检测日期 | `YYYY-MM-DD` 或 `yesterday` |

## 支持的信号类型

| 信号类型 | 说明 | model_opt |
|---------|------|-----------|
| MACD背驰买点 | 价格创新低但MACD未创新低 | 8 |
| MACD背驰卖点 | 价格创新高但MACD未创新高 | 8 |
| 拉回买点 | 价格回调至中枢下沿 | 9 |
| 拉回卖点 | 价格反弹至中枢上沿 | 9 |
| V反买点 | V型反转向上 | 12 |
| V反卖点 | V反转向下 | 12 |

## 检测范围

| 周期 | 检测范围 | 说明 |
|------|----------|------|
| 日线 | 最后 1 根 K 线 | 检测最后一个交易日 |
| 30分钟线 | 最后 8 根 K 线 | 检测最后一个交易日全天（每天约 8 根 30分钟 K 线） |

## 标的类型识别

脚本自动判断持仓标的类型：

| 代码特征 | 类型 | 数据源 |
|---------|------|--------|
| 5xxxxx | 沪市 ETF | `QA_fetch_index_day_adv` |
| 15xxxx | 深市 ETF | `QA_fetch_index_day_adv` |
| 16xxxx | 深市 ETF | `QA_fetch_index_day_adv` |
| 其他 | A 股 | `QA_fetch_stock_day_adv` |
