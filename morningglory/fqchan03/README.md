# 缠论量化通达信插件

## 编译

通达信插件需要编译成32位，下面以Visual Studio 2019举例，作者用的是Visual Studio 2019社区版。

```cmd
mkdir build
cd build
cmake -G "Visual Studio 16 2019" -A Win32 ..
cmake --build . --config Release
```

## 通达信主图代码

```text
BI:=TDXDLL3(1,H,L,0);
NOTEXT画上升笔1:DRAWLINE(BI=-1,L,BI=+1,H,0), DOTLINE, COLORYELLOW;
NOTEXT画下降笔1:DRAWLINE(BI=+1,H,BI=-1,L,0), DOTLINE, COLORYELLOW;

DUAN:=TDXDLL3(2,BI,H,L);
NOTEXT画上升段1:DRAWLINE(DUAN=-1,L,DUAN=+1,H,0), COLORFF8000;
NOTEXT画下降段1:DRAWLINE(DUAN=+1,H,DUAN=-1,L,0), COLORFF8000;

DUANZG:=TDXDLL3(3,DUAN,H,L);
DUANZD:=TDXDLL3(5,DUAN,H,L);
DUANSE:=TDXDLL3(7,DUAN,H,L);

NOTEXT_DDUANZG:IF(DUANZG,DUANZG,DRAWNULL),COLORFF8000;
NOTEXT_DDUANZD:IF(DUANZD,DUANZD,DRAWNULL),COLORFF8000;
NOTEXT_DDUANSE:STICKLINE(DUANSE,DUANZD,DUANZG,0,0),COLORFF8000;

BIZG:=TDXDLL3(3,BI,H,L);
BIZD:=TDXDLL3(5,BI,H,L);
BISE:=TDXDLL3(7,BI,H,L);

NOTEXT_DBIZG:IF(BIZG,BIZG,DRAWNULL),COLORYELLOW;
NOTEXT_DBIZD:IF(BIZD,BIZD,DRAWNULL),COLORYELLOW;
NOTEXT_DBISE:STICKLINE(BISE,BIZD,BIZG,0,0),COLORYELLOW;
```
